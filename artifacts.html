Knowledge Artifacts: Activity Definitions

These define activities that can be performed as part of content in this implementation guide.

Polio Vaccination Session

Administer bivalent oral polio vaccine (bOPV) to all children 0–59 months at a vaccination site.
Rapid Coverage Monitoring

Post-campaign rapid coverage assessment at health facility catchment area level.
Supervision Visit

Supervisory visit to monitor and support campaign execution at the district level.

Knowledge Artifacts: Plan Definitions

These define workflows, rules, strategies, or protocols as part of content in this implementation guide.

Polio SIA Campaign

Campaign template for a Polio Supplementary Immunization Activity targeting children 0–59 months.

Structures: Resource Profiles

These define constraints on FHIR resources for systems conforming to this implementation guide.

Microplan CarePlan

An instantiated immunization campaign for a specific geographic area and time period.
Microplan Location

A location in the immunization campaign hierarchy, from country down to vaccination site.
Microplan PlanDefinition

A campaign template that defines activities and which location hierarchy levels they target.
Microplan Target Population

Estimated target population for vaccination at a specific location.
Microplan Task

A concrete campaign activity assigned to a team at a specific location and time.

Structures: Extension Definitions

These define constraints on FHIR data types for systems conforming to this implementation guide.

Campaign Location

Links a resource to a location in the campaign hierarchy.
Target Location Type

Specifies which level of the location hierarchy a PlanDefinition action targets for Task generation.
